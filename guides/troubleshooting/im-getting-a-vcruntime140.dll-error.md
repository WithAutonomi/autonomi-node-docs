# I'm getting a vcruntime140.dll error

This error can occur when trying to install the Autonomi Node Launchpad on Windows.&#x20;

This typically happens with a new clean Windows install, or image, or when setting up a new computer for the first time.&#x20;

Thankfully there is a simple solution:\
\
[**Install the Microsoft Visual C++**](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170) redistributable.
