title: Any Ethereum address can be...
---

{% hint style="info" %}
Any Ethereum address can be used here, but you'll need to set up a [compatible wallet](https://app.gitbook.com/s/yZv1zkASl6hUNUmP3ypg/using-autonomi-tokens/holding/compatible-wallets) app in order to be able to see and use any tokens you earn. We've got a [step-by-step guide](https://app.gitbook.com/s/yZv1zkASl6hUNUmP3ypg/using-autonomi-tokens/holding/how-to-create-a-metamask-wallet) on how to do that.
{% endhint %}
