# Start a Node

Starting a node (or even several) is easy, and will work on home computers just like yours. Here's how:<br>

{% stepper %}
{% step %}
### **Download the Node Launchpad**&#x20;

[Windows](https://downloads.autonomi.com/launchpad/windows) (Windows 10+)

[Mac](https://downloads.autonomi.com/launchpad/mac) (Apple Silicon)

[Linux x86-64](https://downloads.autonomi.com/launchpad/linux-x64) (Ubuntu 24.04+ recommended)

[Linux ARM](https://downloads.autonomi.com/launchpad/linux-arm)
{% endstep %}

{% step %}
### **Take a look at the Quickstart Guide**&#x20;

Our [step-by-step guide](quickstart-guide/) will walk your through each step of getting nodes running your computer.
{% endstep %}

{% step %}
### **Add your Wallet Address**

Adding your Ethereum Address allows you to hold anything your node earns, but you'll need a [compatible wallet](https://app.gitbook.com/s/yZv1zkASl6hUNUmP3ypg/using-autonomi-tokens/holding/compatible-wallets) to be able to see and use the tokens. The more data your nodes store serve, and the longer and more reliably they do it, the more they will earn, and the more you're rewarded.
{% endstep %}
{% endstepper %}
