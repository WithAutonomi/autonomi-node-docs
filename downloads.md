# Downloads

The easiest way to contribute your spare capacity to the Autonomi Beta Network is with the Node Launchpad. It’s a small application that allows you to run several nodes on your computer that run in the background while you get on with other things.

### Download the Node Launchpad:

[Windows](https://downloads.autonomi.com/launchpad/windows) (Windows 10+ recommended)

[Mac](https://downloads.autonomi.com/launchpad/mac) (Apple Silicon)

[Mac](https://downloads.autonomi.com/launchpad/mac-intel) (Intel)

[Linux x86-64](https://downloads.autonomi.com/launchpad/linux-x64) (Ubuntu 24.04+ recommended)

[Linux ARM](https://downloads.autonomi.com/launchpad/linux-arm)

[Linux ARM 64](https://downloads.autonomi.com/launchpad/linux-arm64)

[Linux ARM v7](https://downloads.autonomi.com/launchpad/linux-armv7)

### **For Power Users**

If you are comfortable with Command Line Interfaces and want to run nodes on headless devices, or on more advanced setups such as virtual environments or remote servers, starting and managing many nodes at once, then try our [antctl](https://github.com/maidsafe/autonomi/tree/main/ant-node-manager). <br>

{% hint style="warning" %}
The Node Launchpad is currently **limited to 50 nodes per device**. If you want to add more than 50 nodes on a single device (and you have the hardware to do it!) then you are probably a _Power User_ who would benefit from using the [antctl](https://github.com/maidsafe/autonomi/tree/main/ant-node-manager).
{% endhint %}
